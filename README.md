# Rachit Sharma – Personal Portfolio 🚀

Hi there! 👋

- Technical and soft skills  
- Selected projects  
- Work experience & leadership roles

- **Frontend**: React.js + Bootstrap  
- **Styling**: Custom CSS, Animate.css  
- **Email Service**: EmailJS  
- **Deployment**: GitHub Pages
